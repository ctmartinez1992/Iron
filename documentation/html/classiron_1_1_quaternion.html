Detailed Description

Quaternions are used to represent an object's orientation (rotation) in space.

They are not affected by gimbal lock and can be interpolated. Used as a replacement for rotation matrices and euler angles.

The class does not handle automatic normalization, the normalize method should be called when necessary.
