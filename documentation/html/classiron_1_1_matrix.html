Detailed Description

Represents a 4 x 4 matrix.

The matrix uses column-major format and its components are organized like an OpenGl translation matrix (compatible) which as the following aspect:

—————————————— ¦ 1 0 0 x ¦ ¦ 0 1 0 y ¦ ¦ 0 0 1 z ¦ ¦ 0 0 0 1 ¦ ——————————————

Multiplication must be done in the the reverse order of operations. Rotation (Matrix R) and then translation (Matrix T) = RT.

If multiple transformations are needed, the class Transform should be used instead of this one.
